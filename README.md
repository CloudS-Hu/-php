二、项目功能介绍
-
该贴吧项目包含如下操作：
  用户注册、登录、注销、发帖、回帖、评论功能以及帖子分页；
三、数据库设计

     用户表：user
     发帖表：post
     回帖表：thread
     评论表：comment
